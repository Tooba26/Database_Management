**Database:**
We'll define some database definitions from different sources so we can understnd the basic meaning of this term.
1) A database is a collection of related information. For example: Phonebook, your 5 best friends and shopping list etc. https://youtu.be/HXV3zeQKqGY

2) A database is a collection of related data. Here data means known facts that can be recorded and have implicit meaning. (Elmasri & Navathe., 2016).
 - A database represents some aspect of the real world, sometimes called the miniworld or the universe of discourse (UoD). Changes to the miniworld
are reflected in the database.
 - A database is a logically coherent collection of data with some inherent meaning. A random assortment of data cannot correctly be referred to as a database.
 - A database is designed, built, and populated with data for a specific purpose. It has an intended group of users and some preconceived applications in which these users are interested.
